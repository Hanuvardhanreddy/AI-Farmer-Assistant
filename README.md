# 🌾 AI Farmer Assistant

An AI-powered web application that helps farmers access agricultural information through a modern and easy-to-use interface.

---

## 📌 Features

- 🤖 AI Chatbot
- 🌱 Farming Assistance
- 📱 Responsive Design
- ⚡ Fast Performance
- 🎨 Modern UI

---

## 🛠 Technologies Used

- React.js
- Vite
- JavaScript
- HTML5
- CSS3

---

## 📂 Folder Structure

```
src/
public/
package.json
vite.config.js
```

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Hanuvardhanreddy/AI-Farmer-Assistant.git
```

### Navigate to the project

```bash
cd AI-Farmer-Assistant
```

### Install dependencies

```bash
npm install
```

### Start the application

```bash
npm run dev
```
